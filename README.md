## Tech Stack

- **Frontend**: nextjs
- **Backend**: nodejs
- **Database**: sqlite
- **ORM**: better-sqlite3
- **Auth**: nextauth

## Getting Started

```bash
npm install

# Setup database (SQLite by default - no external DB needed)
npx prisma generate
npx prisma db push

# Start development server
npm run dev
```

### Database Commands

```bash
# Push schema changes to database
npx prisma db push

# Create a migration
npx prisma migrate dev --name init

# Open Prisma Studio (database GUI)
npx prisma studio

# Reset database
npx prisma migrate reset
```
